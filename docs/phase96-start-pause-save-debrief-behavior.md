# Phase 96 Pass 1C - Start/Pause/Save/Debrief Behavior

The Mission Active controls now update a local draft:

- Start opens or resumes the draft.
- Pause marks the draft paused.
- Save Checkpoint marks a checkpoint.
- Debrief closes the draft preview for review.

The controls still do not write to Notion, Sheets, Drive, app ledgers, mission completion, XP, or notifications.
