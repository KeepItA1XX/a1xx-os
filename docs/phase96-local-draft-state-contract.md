# Phase 96 Pass 1B - Local Draft State Contract

The local draft state has one compact time-session shape.

Required fields:

- draft id
- status
- start and end timestamps
- timezone
- mission id and title
- duration minutes
- paused minutes
- checkpoint count
- debrief count
- proof, result, and debrief placeholders
- approval hold

The draft is in-memory only for this phase.
